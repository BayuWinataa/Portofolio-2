# Portfolio

Sebuah portofolio modern yang dibangun dengan Next.js, TypeScript, dan Tailwind CSS. Menampilkan proyek, pengalaman, dan desain UI/UX; dilengkapi animasi dan support dark mode.

**Teknologi**

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons
